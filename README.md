# info2180-finalproject

Setup:
1. Download repo in C:\\xampp
2. Open XAMP Control Panel and start Apache and MySQL
2. Copy the contents on schema.sql into SQL editor for XAMPP to create datebase and default user

Run:
1. Open [localhost/xamp](http://localhost/info2180-finalproject/)
2. Log in with account ( admin@project2.com / password123)
